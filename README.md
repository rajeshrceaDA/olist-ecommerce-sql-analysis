#  End-to-End Olist E-commerce Sales Analysis using SQL & Power BI

##  Project Overview

This project demonstrates an end-to-end SQL data analysis workflow using the Olist Brazilian E-commerce dataset.

The project focuses on solving real business problems through SQL by performing:

- Data Cleaning
- Data Validation
- Exploratory Data Analysis (EDA)
- Business KPI Analysis
- Customer Analysis
- Product Performance Analysis
- Sales Analysis
- Power BI Dashboard Creation

The objective is to simulate the daily responsibilities of a Data Analyst and generate actionable business insights for decision-making.

## Business Questions Solved

This project answers the following business questions:

1. What is the total revenue generated?
2. How many total orders were placed?
3. What is the Average Order Value (AOV)?
4. Which product categories generate the highest revenue?
5. Which states have the highest sales?
6. Which payment methods are most frequently used?
7. Which sellers generate the highest revenue?
8. How is revenue changing month by month?
9. How long does delivery take on average?
10. What business insights can improve sales performance?

## Key Performance Indicators (KPIs)

The project measures the following business KPIs:

- Total Revenue
- Total Orders
- Average Order Value (AOV)
- Total Customers
- Total Sellers
- Top Product Categories
- Monthly Revenue Trend
- Monthly Order Trend
- Average Delivery Time
- Payment Method Distribution
